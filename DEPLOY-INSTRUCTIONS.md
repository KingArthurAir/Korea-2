## ⚠️ 重要提醒

### 关于 AB 页/斗篷
**再次提醒**: Google Ads 严格禁止 Cloaking(斗篷) 行为

- 如果被发现向审核人员和真实用户展示不同页面
- **后果**: 广告账户永久封禁 + 域名黑名单 + 关联账户连坐

### 建议做法
1. **使用单一合规页面** (即本文件)
2. **确保页面内容与广告一致**
3. **真实填写公司信息**
4. **保留所有政策页面**

## 🔧 可选自定义

### 修改颜色
在 CSS 中修改主色调:
```css
/* 主色调 - 当前为蓝紫色 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 品牌色 - 当前为深青色 */
color: #1a5f7a;
```

### 修改统计数据
在 About 部分修改数字:
```html
<div class="stat-number">32M+</div>  <!-- 改为你真实的用户数 -->
<div class="stat-number">98%</div>   <!-- 改为你真实的满意度 -->
```

### 添加真实图片
替换占位图片:
```html
<img src="your-real-image.jpg" alt="描述">
```
